# 混合式版本管理器测试指南

## 概述

混合式版本管理器结合了**下拉菜单**（快速访问）和**弹窗对话框**（完整管理）两种交互方式。

## 测试步骤

### 1. 启动测试

运行以下命令启动 launcher：

```bash
python launcher/main.py
```

或使用测试脚本：

```bash
test-hybrid-version-manager.bat
```

### 2. 测试下拉菜单功能

#### 步骤：
1. 找到窗口顶部的 **"📦 版本管理"** 按钮（蓝色按钮）
2. 点击该按钮

#### 预期结果：
- ✅ 显示下拉菜单
- ✅ 菜单包含最近 5 个版本（如果有）
- ✅ 每个版本显示：
  - 短 commit hash（如 `2b81b9a`）
  - commit 说明（前 40 个字符）
- ✅ 鼠标悬停时显示完整 tooltip（完整说明 + 日期）
- ✅ 底部有分隔线和 "📋 查看更多版本..." 选项

### 3. 测试快速切换功能

#### 步骤：
1. 在下拉菜单中点击任意版本

#### 预期结果：
- ✅ 弹出确认对话框
- ✅ 显示版本信息：
  - Commit hash
  - 完整说明
  - 日期
- ✅ 点击"是"后显示"正在切换版本..."提示
- ✅ 切换完成后显示成功消息
- ✅ 程序自动退出（需要手动重启）

### 4. 测试完整弹窗功能

#### 步骤：
1. 点击下拉菜单中的 **"📋 查看更多版本..."** 选项

#### 预期结果：
- ✅ 打开完整的版本管理器弹窗
- ✅ 显示当前版本信息（绿色边框）
- ✅ 显示可用版本列表（最多 20 个）
- ✅ 每个版本显示：
  - Commit hash（红色）
  - 完整说明
  - 日期
  - "[当前使用]" 标记（如果是当前版本）
- ✅ 每个版本有操作按钮：
  - "👁️ 预览" - 查看详细说明
  - "🔄 切换到此版本" - 切换到该版本（当前版本除外）
- ✅ 底部有功能按钮：
  - "🔄 刷新" - 重新加载版本列表
  - "📝 查看完整日志" - 查看最近 50 条 commit
  - "关闭" - 关闭弹窗

### 5. 测试边界情况

#### 测试 5.1: Git 不可用
- 临时重命名 `.git` 文件夹为 `.git.bak`
- 点击版本管理按钮
- 预期：直接打开完整弹窗，显示"无法获取当前版本"

#### 测试 5.2: 无版本历史
- 在新初始化的 Git 仓库中测试
- 预期：显示友好提示信息

#### 测试 5.3: 大量版本
- 在有超过 20 个 commit 的仓库中测试
- 预期：下拉菜单显示最近 5 个，弹窗显示最近 20 个

## 常见问题排查

### 问题 1: 点击按钮无反应
**可能原因：**
- Git 未安装或不在 PATH 中
- PyQt6 未正确安装

**解决方法：**
```bash
# 检查 Git
where git

# 检查 PyQt6
python -c "from PyQt6.QtWidgets import QApplication"
```

### 问题 2: 版本列表为空
**可能原因：**
- 项目不是 Git 仓库
- `.git` 文件夹损坏

**解决方法：**
```bash
# 检查是否在 Git 仓库中
git status

# 初始化 Git（如果需要）
git init
git add .
git commit -m "Initial commit"
```

### 问题 3: 切换版本失败
**可能原因：**
- 工作区有未提交的更改
- Git checkout 命令执行失败

**解决方法：**
- 提交或暂存当前更改
- 检查 Git 权限
- 查看详细错误信息

在 `launcher/main.py` 中添加调试输出：

```python
def _get_recent_versions(self, limit=5):
    try:
        import subprocess
        print(f"[DEBUG] 获取版本列表，limit={limit}")
        result = subprocess.run(
            ['git', 'log', f'-n {limit}', '--pretty=format:%h|%s|%ai'],
            capture_output=True, text=True,
            cwd=self.base_dir, timeout=5
        )
        print(f"[DEBUG] Git 返回：{result.stdout[:200]}")

```
