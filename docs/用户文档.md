# 云集智能音乐创意台 - 用户文档

## 项目简介

云集智能音乐创意台是一款基于 [ACE-Step 1.5](https://github.com/ace-step/ACE-Step-1.5) 开源 AI 音乐生成模型的本地化音乐创作平台。它提供类似 Spotify 的专业级界面，让您在本地 GPU 上免费、无限量地生成高质量 AI 音乐，无需云端订阅。

### 核心优势

| 对比项 | Suno / Udio | 云集智能音乐创意台 |
|--------|-------------|-------------------|
| 费用 | $10-50/月 | **完全免费** |
| 隐私 | 云端处理 | **100% 本地运行** |
| 版权 | 受限许可 | **作品完全归您所有** |
| 自定义 | 有限 | **完全可控** |
| 生成限制 | 有排队限制 | **无限制** |
| 商用 | 需高级套餐 | **无任何限制** |

## 系统要求

| 要求 | 规格 |
|------|------|
| **Node.js** | 18 或更高版本 |
| **Python** | 3.10+（推荐 3.11） |

## 安装指南

### 方式一：Windows 便携包（推荐）

1. 下载 [ACE-Step-1.5.7z](https://files.acemusic.ai/acemusic/win/ACE-Step-1.5.7z)（约 5GB）
2. 解压到 `C:\ACE-Step-1.5`（或您喜欢的位置）
3. 便携包已包含 `python_embeded` 及所有依赖，无需额外安装

> 4GB 显存即可运行（LLM 功能自动禁用）。12GB+ 显存可启用 Thinking Mode。

### 方式二：标准安装（全平台）

```bash
git clone https://github.com/ace-step/ACE-Step-1.5
cd ACE-Step-1.5
uv venv
uv pip install -e .
# 模型会在首次运行时自动下载（约 5GB）
```

### 安装前端 UI

```bash
git clone https://github.com/fspecii/ace-step-ui
cd ace-step-ui

# Linux / macOS
./setup.sh

# Windows
setup.bat
```

手动安装：

```bash
npm install
cd server && npm install && cd ..
cp server/.env.example server/.env   # Linux/macOS
copy server\.env.example server\.env  # Windows
```

## 快速启动

### Windows 一键启动

```batch
cd ace-step-ui
start-all.bat
```

> 默认查找 ACE-Step 在 `..\ACE-Step-1.5`。如路径不同，请先设置：
> ```batch
> set ACESTEP_PATH=C:\path\to\ACE-Step-1.5
> start-all.bat
> ```

### Linux / macOS 一键启动

```bash
cd ace-step-ui
./start-all.sh
```

停止服务：`./stop-all.sh`

### 手动启动

**第一步：启动 ACE-Step API 服务**

```batch
REM Windows 便携包
cd C:\ACE-Step-1.5
python_embeded\python acestep\api_server.py
```

```bash
# Linux / macOS / Windows 标准安装
cd /path/to/ACE-Step-1.5
uv run acestep-api --port 8001
```

等待出现 "Application startup complete" 后继续。

**第二步：启动前端 UI**

```bash
# Linux / macOS
cd ace-step-ui && ./start.sh

# Windows
cd ace-step-ui && start.bat
```

**第三步：打开浏览器**

| 访问方式 | 地址 |
|---------|------|
| 本机 | http://localhost:3000 |
| 局域网其他设备 | http://您的IP:3000 |

## 功能详解

### 1. AI 音乐生成

#### 简单模式

只需用自然语言描述您想要的音乐，AI 会自动处理其余部分。

> 示例："一首关于夏日冒险的欢快流行歌曲，带有朗朗上口的旋律"

#### 自定义模式

完全控制每一个生成参数：

| 参数 | 说明 |
|------|------|
| **歌词** | 支持使用 `[Verse]`、`[Chorus]` 等结构标签 |
| **风格** | 定义流派、情绪、乐器、速度 |
| **时长** | 30-240 秒 |
| **BPM** | 60-200 节拍/分钟 |
| **调性** | C major、A minor 等 |
| **拍号** | 2/4、3/4、4/4、6/8 |

#### 批量生成

| 设置 | 说明 |
|------|------|
| **Batch Size** | 每次任务生成的变体数量（1-4），默认为 1。8GB 显存建议保持为 1 |
| **Bulk Generate** | 排队多个独立生成任务（1-10），按顺序执行，对任何 GPU 都安全 |

### 2. 风格系统

内置 **936 种风格**，同步自 Suno 的探索器，支持搜索和随机选择。涵盖：

- 一级分类：流行、摇滚、电子、古典、爵士、R&B、嘻哈、民谣等
- 二级分类：每个一级分类下的细分子风格

### 3. 高级参数

| 功能 | 说明 |
|------|------|
| **参考音频** | 使用任意音频文件作为风格参考 |
| **音频翻唱** | 用新风格转换已有音频 |
| **局部重绘** | 重新生成曲目的特定片段 |
| **种子控制** | 精确复现某次生成结果 |
| **推理步数** | 控制质量与速度的平衡 |
| **Thinking Mode** | 让 AI 自动增强您的提示词（需 12GB+ 显存） |

### 4. LoRA / LoKR 训练

支持自定义音色训练，打造专属音乐风格：

- **LoRA 训练**：读取文件夹中的音频数据，显存优化，最低显存即可训练
- **LoKR 训练**：减少训练时间，提升音质
- **数据集构建器**：可视化创建和管理训练数据集
- **加载适配器**：LoRA 读取文件夹，LoKR 读取 safetensors 文件

训练流程：
1. 准备音频数据集
2. 标注歌词和元数据（BPM、调性等）
3. 配置训练参数
4. 开始训练
5. 加载训练好的适配器进行生成

### 5. 内置工具

| 工具 | 说明 |
|------|------|
| **音频编辑器** | 基于 AudioMass，支持裁剪、淡入淡出、效果处理 |
| **人声分离** | 基于 Demucs，分离人声、鼓、贝斯和其他音轨 |
| **视频生成器** | 为音乐创建可视化视频，支持多种预设效果 |
| **专辑封面** | 自动生成渐变风格封面，无需联网 |

#### 视频生成器预设

提供 10 种可视化预设：NCS Circle、Linear Bars、Dual Mirror、Center Wave、Orbital、Digital Rain、Hexagon、Shockwave、Oscilloscope、Minimal。

支持特效：抖动、故障风、VHS、CCTV、扫描线、色差、光晕、胶片颗粒、像素化、频闪、暗角、色相偏移。

### 6. 音乐库管理

- **浏览与搜索**：快速查找所有曲目
- **收藏**：标记喜欢的歌曲
- **播放列表**：创建自定义播放列表，组织音乐
- **歌曲详情**：查看完整的生成参数，一键复用

### 7. 专业播放器

- 底部全功能播放器
- 波形可视化
- 进度条拖拽定位
- 音量控制

## 配置说明

编辑 `server/.env` 文件：

```env
# 服务端口
PORT=3001

# ACE-Step API 地址
ACESTEP_API_URL=http://localhost:8001

# 数据库路径（本地优先，无需云端）
DATABASE_PATH=./data/acestep.db

# 可选：Pexels API 密钥（用于视频背景）
PEXELS_API_KEY=your_key_here
```

## 常见问题

| 问题 | 解决方案 |
|------|---------|
| ACE-Step API 无法连接 | 确保 API 服务已启动（参见启动步骤） |
| CUDA 显存不足 | 切换 LM Backend 为 **PT**，Batch Size 设为 **1**，缩短时长，或关闭 Thinking Mode |
| 4GB 显存 - 显存不足 | 使用 **PT** 后端（默认），Batch Size **1**，保持 Thinking Mode **关闭** |
| 歌曲时长显示 0:00 | 安装 FFmpeg：Linux 用 `sudo apt install ffmpeg`，Windows 从 [ffmpeg.org](https://ffmpeg.org) 下载 |
| 局域网无法访问 | 检查防火墙是否允许 3000 和 3001 端口 |
| AttributeError: 'NoneType' | 更新到最新版 ACE-Step-1.5 |
