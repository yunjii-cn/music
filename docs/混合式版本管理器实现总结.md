# 混合式版本管理器实现总结

## 项目概述

**实现时间：** 2026-03-19  
**需求来源：** 用户要求 "用混合式吧" - 结合下拉菜单和弹窗两种交互方式  
**设计理念：** 效率与功能完整性并重

## 核心特性

### 1. 混合式设计 (Hybrid Design)

```
点击 "📦 版本管理" 按钮
         ↓
    ┌─────────────┐
    │  下拉菜单   │ ← 快速访问（效率优先）
    │ 最近 5 个版本 │
    ├─────────────┤
    │ 查看更多... │ → 完整弹窗（功能完整）
    └─────────────┘
```

### 2. 双模式交互

| 模式 | 触发方式 | 显示内容 | 适用场景 |
|------|----------|----------|----------|
| **快速模式** | 下拉菜单 | 最近 5 个版本 | 切换到最近的版本 |
| **完整模式** | 点击"查看更多..." | 最近 20 个版本 + 搜索/过滤 | 浏览历史、查找特定版本 |

# 获取最近 5 个版本
result = subprocess.run(

    capture_output=True, text=True,
    cwd=self.base_dir, timeout=5

# 解析格式：hash|message|date
for line in result.stdout.strip().split('\n'):
    parts = line.split('|')
    versions.append({

        'date': parts[2][:16]  # 截取前 16 个字符
    })

#### 3. 用户数据保护机制

```python
# 备份目录
user_dirs = ['config', 'models', 'output', '.env']
backup_dir = os.path.join(self.base_dir, '.git_backup')

# 备份过程
for dir_name in user_dirs:
    src = os.path.join(self.base_dir, dir_name)
    dst = os.path.join(backup_dir, dir_name)
    if os.path.exists(src):
        if os.path.isdir(src):
            shutil.copytree(src, dst)

    if not versions:
        # 降级方案：直接打开完整弹窗

        return

```
