### 6. 修复青龙训练器 API 服务启动失败（06:37）

- **修复**：在 `dev/data/.venv` 中安装 `socksio==1.0.0`
- **验证**：httpx 0.28.1 + Gradio 6.2.0 在 SOCKS 代理环境下初始化正常
