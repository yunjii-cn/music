## 📋 必须记住的重要原则

### ❌ 绝对不能做的事

1. **不要直接删除文件** - 所有多余文件都移动到 `BAK/` 文件夹
2. **不要创建新的打包脚本** - 只使用 `build-correct.py`
3. **不要修改多个启动脚本** - 只修改明确要求的那个
4. **不要跳过文档直接编码** - 先看文档再动手

### ✅ 必须做的事

1. **先阅读本文档** - 特别是 `.ai-context/QUICKSTART.md`
2. **使用标准打包脚本** - `build-correct.py`
3. **在main分支开发** - 日常开发在main分支
4. **测试后合并到beta** - 测试通过后合并到beta分支
5. **稳定后合并到stable** - 充分测试后才合并到stable

## 📁 BAK文件夹使用指南

**重要原则**: 不直接删除，而是归档

### BAK子文件夹说明

| 子文件夹 | 存放内容 |
|---------|---------|
| `packaging/` | 废弃的打包脚本 |
| `launchers/` | 废弃的启动脚本 |
| `temp_scripts/` | 废弃的测试和临时脚本 |
| `temp_python/` | 废弃的临时Python文件 |
| `temp_docs/` | 废弃的文档和临时文件 |
| `launcher_backups/` | launcher目录中的备份文件 |
| `scripts_temp/` | scripts目录中的废弃文件 |
| `utility_scripts/` | 整理工具脚本 |
| `misc_files/` | 其他零散文件 |
| `extra_configs/` | 额外配置文件 |
| `extra_docs/` | 额外文档 |
| `old_exes/` | 旧的EXE文件 |
| `linux_scripts/` | Linux脚本 |
| `extra_folders/` | 额外文件夹 |
