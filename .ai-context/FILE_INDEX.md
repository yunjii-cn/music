## 启动器模块

### `launcher/main.py`
**用途**: PyQt6 GUI启动器主程序  
**核心功能**:
- 环境维护与检测
- 模型管理界面
- 项目服务启动（青龙训练器、官方音乐演练场等）
- 版本管理器集成
- 日志显示和UI更新

**关键类**:
- `MainWindow`: 主窗口类
- `ServiceManager`: 服务管理器
- `EnvironmentMaintenance`: 环境维护线程

**修改注意事项**:
- 使用信号槽机制进行线程安全的UI更新
- 不要直接修改UI元素，必须通过信号
- 保持代码模块化

---

### `launcher/version_manager.py`
**用途**: Git版本管理器  
**核心功能**:
- Git分支切换
- 版本历史查看
- 版本切换界面

## 打包工具

### `build-correct.py`
**用途**: 标准EXE打包脚本（唯一推荐使用）  
**核心功能**:
1. 清理构建目录
2. 使用PyInstaller构建单文件EXE
3. 检测Git更改
4. 生成详细提交信息
5. 自动提交到Git
6. 自动推送到当前分支

**输出位置**: `dist/云集智能音乐创意台-vYYYY.MM.DD.HHMM.exe`

**使用方式**:
```bash
python build-correct.py
```

**修改注意事项**:
- 这是唯一的标准打包脚本
- 不要创建新的打包脚本
- 所有修改应在此脚本中进行

## 文件夹说明

### `BAK/`
**用途**: 遗留文件归档文件夹  
**重要**: 所有多余文件都应移动到这里，不要直接删除  
**子文件夹**:
- `packaging/`: 废弃的打包脚本
- `launchers/`: 废弃的启动脚本
- `temp_scripts/`: 废弃的测试和临时脚本
- `temp_python/`: 废弃的临时Python文件
- `temp_docs/`: 废弃的文档和临时文件
- `launcher_backups/`: launcher目录中的备份文件
- `scripts_temp/`: scripts目录中的废弃文件
- `utility_scripts/`: 整理工具脚本
- `misc_files/`: 其他零散文件
- `extra_configs/`: 额外配置文件
- `extra_docs/`: 额外文档
- `old_exes/`: 旧的EXE文件
- `linux_scripts/`: Linux脚本
- `extra_folders/`: 额外文件夹
